# Contenu final — Archive L2 Maths

Structure pensée pour réviser : **Cours → TD → Tests/CC → Partiels → Examens → Corrigés**.

## Règles appliquées

- Les documents **eCampus 2025-2026** sont prioritaires pour les cours, notes de cours, TD et évaluations de l’année.
- Le **Drive étudiant** est utilisé comme complément, archive ancienne, ou pour récupérer les documents absents d’eCampus.
- Exception : **Maths approfondies S2 / Graphes**. Comme il n’y a pas de vrai cours officiel complet, les notes du Drive et la traduction Bondy–Murty sont conservées comme contenu central.
- Les catégories **Administratif**, **Résultats / notes** et **Anglais** sont conservées dans le manifest mais cachées du site mathématique.
- Les doublons exacts sont masqués automatiquement : une seule version visible reste dans le contenu final.
